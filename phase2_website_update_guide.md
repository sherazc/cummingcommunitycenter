# Website Update Playbook (Phase 2)

Target page: https://cummingcommunitycenter.com/home/

## Scope
Update only the content on /home/ to match Flyer-phase2.md.

## Platform Details
- WordPress 6.9.4
- PHP 8.2
- Theme: Divi 5.7.1
- DB: MySQL 8.2

## Important Constraints
- Keep the existing URL and one-page structure.
- Use Flyer-phase2.md as the source of truth.
- Remove old Phase 1 investment share and ROI messaging.

## Click-by-Click Divi Steps
1. Log in to WP Admin.
2. Open Pages > Home (the page used by /home/).
3. Click Edit With Divi.
4. Replace text modules in the order below.
5. Remove old sections that contain:
   - Share count, ROI, and investment duration
   - Old phase breakdown and old FAQ related to share model
   - Townhall/presentation blocks if they are tied to old messaging
6. Save Draft.
7. Preview on desktop and mobile.
8. Publish.
